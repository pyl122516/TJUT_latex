\# TJUT

本项目是天津理工大学(2023)工科硕士毕业论文，latex模板

欢迎使用 TUTthesis！

本项目旨在创建**天津理工大学学位论文**的 LaTeX 模板，为天津理工大学学生的论文撰写提供便利。

\#\# 使用

1. 克隆或下载本仓库
2. 使用 XeLaTeX 编译主文件
3. 按需修改正文内容

**功能特点**

- 符合天津理工大学学位论文格式要求
- 结构清晰，易于上手
- 持续维护更新
